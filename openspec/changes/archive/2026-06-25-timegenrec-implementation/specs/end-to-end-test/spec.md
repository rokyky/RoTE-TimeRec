## 新增需求

### 需求: 合成数据 pipeline 测试
系统必须有 pytest，在小规模合成数据（100 用户，50 物品）上运行完整 pipeline，不依赖外部下载。

#### 场景: 合成数据测试完成
- **当** 执行端到端测试时
- **那么** pipeline 完成执行并返回非空 CandidateList

### 需求: 训练+评估往返测试
系统必须验证模型在合成数据上训练 2 epoch 后产生非 NaN 评估指标。

#### 场景: 训练评估有效
- **当** 模型在合成数据上训练 2 epoch 后评估
- **那么** 评估指标为有限数值且不为 NaN
